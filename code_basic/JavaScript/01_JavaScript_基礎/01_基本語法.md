# JavaScript 基本語法

## 什麼是語法？

語法是程式語言的規則集合，它定義了如何正確地書寫程式碼，以便程式語言的解釋器或編譯器能夠理解並執行。JavaScript 語法與許多 C 語言家族的語言相似，但也有其獨特之處。

## 1. 語句 (Statements)

JavaScript 程式是由一系列語句組成的。每個語句都是一個指令，用於執行某種操作。

```javascript
// 這是一個語句，用於在控制台輸出內容
console.log("Hello, JavaScript!");

// 這也是一個語句，宣告一個變數並賦值
let message = "歡迎學習 JavaScript";
```

### 分號 (Semicolons)

在 JavaScript 中，語句結束時使用分號 `;` 是可選的，但普遍建議使用，特別是對於初學者，這有助於避免自動分號插入 (Automatic Semicolon Insertion, ASI) 可能導致的潛在問題，並提升程式碼的可讀性。

```javascript
let name = "Alice"; // 建議加上分號
let age = 30;      // 建議加上分號

// 沒有分號也能正常執行，但有時可能引發問題
// let name = "Alice"
// let age = 30
```

## 2. 註解 (Comments)

註解是程式碼中被 JavaScript 引擎忽略的部分，主要用於解釋程式碼的功能或提供備註，提升程式碼的可維護性。

### 單行註解

使用雙斜線 `//`，從 `//` 到行尾的內容都會被視為註解。

```javascript
// 這是一條單行註解
let x = 10; // 這是變數 x 的定義
```

### 多行註解

使用 `/* */`，介於 `/*` 和 `*/` 之間的所有內容都會被視為註解。

```javascript
/*
這是一段多行註解。
它可以跨越多行，
用於解釋更複雜的程式碼區塊。
*/
let y = 20;
```

## 3. 區塊 (Blocks)

JavaScript 語句可以組合到區塊中。區塊由大括號 `{}` 包圍。常用於條件語句 (`if/else`)、迴圈 (`for/while`) 和函數。

```javascript
if (true) {
    // 程式碼區塊開始
    console.log("條件為真");
    let result = 1 + 1;
    console.log("結果是：" + result);
    // 程式碼區塊結束
}
```

## 4. 大小寫敏感 (Case Sensitivity)

JavaScript 是一種大小寫敏感的語言。這意味著 `myVariable` 和 `myvariable` 是兩個不同的變數。

```javascript
let myVariable = "Hello";
let MyVariable = "World";

console.log(myVariable); // 輸出: Hello
console.log(MyVariable); // 輸出: World
```

## 5. 識別字 (Identifiers) 和命名規則

識別字是用於命名變數、函數、類別等的名稱。

### 命名規則：

*   必須以字母 (a-z, A-Z)、底線 (`_`) 或美元符號 (`$`) 開頭。
*   後續字元可以是字母、數字 (0-9)、底線或美元符號。
*   不能使用 JavaScript 的保留字 (Reserved Words) 或關鍵字 (Keywords)。
*   不能包含空格。

### 命名慣例：

*   **camelCase (駝峰式命名)**：第一個單字小寫，後續單字首字母大寫。常用於變數和函數名 (`myVariableName`, `calculateTotal`)。
*   **PascalCase (帕斯卡命名法)**：每個單字的首字母都大寫。常用於類別名 (`MyClassName`)。
*   **UPPER_CASE_SNAKE_CASE**：所有字母大寫，單字間用底線分隔。常用於常數 (`MAX_LENGTH`)。

```javascript
let userName = "John Doe";      // camelCase for variables
function getUserInfo() {        // camelCase for functions
    // ...
}
class CarModel {                // PascalCase for classes
    // ...
}
const PI = 3.14159;             // UPPER_CASE_SNAKE_CASE for constants
```

## 6. 字面量 (Literals)

字面量是程式碼中固定不變的值。

*   **數字字面量 (Number Literals)**：整數或浮點數。
    ```javascript
    let integer = 100;
    let float = 3.14;
    ```
*   **字串字面量 (String Literals)**：用單引號 `' '`、雙引號 `" "` 或反引號 `` ` `` 包圍的文字。反引號可用於模板字面量 (Template Literals)，支援嵌入表達式和多行字串。
    ```javascript
    let singleQuote = 'Hello';
    let doubleQuote = "World";
    let templateLiteral = `The sum is ${1 + 2}.`; // 支援嵌入表達式
    let multiLine = `這是
一個
多行字串`;
    ```
*   **布林字面量 (Boolean Literals)**：`true` 或 `false`。
    ```javascript
    let isStudent = true;
    let hasLicense = false;
    ```
*   **物件字面量 (Object Literals)**：用 `{}` 定義鍵值對集合。
    ```javascript
    let person = { name: "Alice", age: 25 };
    ```
*   **陣列字面量 (Array Literals)**：用 `[]` 定義元素的有序集合。
    ```javascript
    let colors = ["red", "green", "blue"];
    ```

## 7. 關鍵字與保留字 (Keywords and Reserved Words)

JavaScript 有一些特殊意義的單字，不能用作識別字。例如 `if`, `for`, `function`, `let`, `const` 等。

## 8. 基本輸出方式

### `console.log()`

最常用於在瀏覽器開發者工具的控制台或 Node.js 環境中輸出訊息，便於除錯。

```javascript
console.log("這會顯示在控制台");
console.log(123);
let data = { id: 1, name: "Test" };
console.log(data);
```

### `alert()`

在瀏覽器中彈出一個帶有訊息的對話框。這會阻礙使用者與頁面的互動，因此在現代網頁開發中較少使用。

```javascript
// alert("這是一個彈出視窗訊息！");
```

### `document.write()`

直接將內容寫入 HTML 文件。通常在頁面加載完成後使用會覆蓋現有內容，故在實際開發中應謹慎使用。

```javascript
// document.write("<h1>這是由 JavaScript 寫入的標題</h1>");
```
