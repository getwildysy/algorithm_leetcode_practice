# JavaScript 運算子 (Operators)

運算子是用於對變數和值執行操作的符號。

## 1. 算術運算子 (Arithmetic Operators)

用於執行數學運算。

```javascript
let x = 10;
let y = 3;

console.log(x + y);  // 13 (加)
console.log(x - y);  // 7  (減)
console.log(x * y);  // 30 (乘)
console.log(x / y);  // 3.333... (除)
console.log(x % y);  // 1  (取餘數，Modulus)
console.log(x ** y); // 1000 (指數，10的3次方，ES2016)

x++; // 遞增 (Increment)，x 變 11
y--; // 遞減 (Decrement)，y 變 2
```

> **注意**：`+` 運算子如果遇到字串，會進行**字串串接 (Concatenation)**。
> `console.log("10" + 5); // "105"`

## 2. 賦值運算子 (Assignment Operators)

用於將值賦給變數。

```javascript
let a = 10;
a += 5; // 等同於 a = a + 5 (a 變 15)
a -= 2; // 等同於 a = a - 2
a *= 2; // 等同於 a = a * 2
a /= 2; // 等同於 a = a / 2
```

## 3. 比較運算子 (Comparison Operators)

用於比較兩個值，回傳 `true` 或 `false`。**強烈建議使用嚴格比較 (`===`, `!==`)**。

*   `==` (寬鬆相等)：會自動轉換型別 (Type Coercion) 後再比較，容易產生意想不到的結果。
*   `===` (嚴格相等)：不會轉換型別，型別與值都要相同才為真。

```javascript
console.log(5 == "5");  // true  (字串 "5" 被轉為數字 5)
console.log(5 === "5"); // false (型別不同：number vs string)

console.log(10 != 5);   // true
console.log(10 !== "10"); // true

console.log(10 > 5);    // true
console.log(10 <= 10);  // true
```

## 4. 邏輯運算子 (Logical Operators)

用於布林邏輯判斷。

```javascript
let isAdult = true;
let hasTicket = false;

// && (AND): 兩者皆為真才為真
console.log(isAdult && hasTicket); // false

// || (OR): 其中之一為真即為真
console.log(isAdult || hasTicket); // true

// ! (NOT): 反轉布林值
console.log(!isAdult); // false
```

### 短路求值 (Short-circuit evaluation)
*   `A && B`：如果 A 為假，直接回傳 A，不執行 B。
*   `A || B`：如果 A 為真，直接回傳 A，不執行 B。

## 5. 三元運算子 (Ternary Operator)

唯一需要三個運算元的運算子，常用於簡化 `if...else`。

```javascript
// 語法: 條件 ? 值1 (真時回傳) : 值2 (假時回傳)

let age = 20;
let status = (age >= 18) ? "成年" : "未成年";
console.log(status); // "成年"
```
