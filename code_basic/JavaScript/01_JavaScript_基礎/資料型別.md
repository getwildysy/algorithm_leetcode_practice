# JavaScript 資料型別 (Data Types)

JavaScript 是一種**動態型別 (Dynamically Typed)** 語言，這意味著變數本身沒有型別，值才有型別，且變數可以隨時持有不同型別的值。

JavaScript 的資料型別分為兩大類：**基本型別 (Primitives)** 和 **物件型別 (Objects)**。

## 1. 基本型別 (Primitives)

基本型別是**不可變 (Immutable)** 的值，直接儲存值本身。

1.  **String (字串)**：文字資料。
    ```javascript
    let name = "Alice";
    let greeting = `Hello, ${name}`; // 樣板字面量
    ```
2.  **Number (數字)**：JavaScript 只有一種數值型別，包含整數和浮點數。
    ```javascript
    let age = 25;
    let price = 19.99;
    let infinity = Infinity;
    let notANumber = NaN; // Not a Number，例如 "abc" / 2 的結果
    ```
3.  **Boolean (布林)**：邏輯值，只有 `true` 和 `false`。
    ```javascript
    let isLogin = true;
    ```
4.  **Null**：代表「刻意設定為空值」或「無值」。
    ```javascript
    let selectedUser = null;
    ```
5.  **Undefined**：代表「變數已宣告但尚未賦值」或「物件屬性不存在」。
    ```javascript
    let x;
    console.log(x); // undefined
    ```
6.  **Symbol** (ES6)：唯一且不可變的值，常用於物件屬性鍵，避免命名衝突。
7.  **BigInt** (ES2020)：用於安全地儲存和操作大整數 (大於 `2^53 - 1`)。
    ```javascript
    let bigNumber = 1234567890123456789012345678901234567890n;
    ```

## 2. 物件型別 (Objects)

物件是**參考型別 (Reference Type)**，變數儲存的是記憶體位址 (Reference)。修改物件內容會影響所有指向該位址的變數。

*   **Object**：鍵值對 (Key-Value) 的集合。
*   **Array**：有序的列表 (也是一種特殊的 Object)。
*   **Function**：可執行的程式碼區塊 (也是一種特殊的 Object)。
*   **Date**, **RegExp**, **Map**, **Set** 等。

```javascript
let person = { name: "Bob", age: 30 };
let numbers = [1, 2, 3];
```

## 3. typeof 運算子

使用 `typeof` 可以檢查值的型別。

```javascript
typeof "Hello"   // "string"
typeof 123       // "number"
typeof true      // "boolean"
typeof undefined // "undefined"
typeof Symbol()  // "symbol"
typeof 10n       // "bigint"

// 注意事項：
typeof {a: 1}    // "object"
typeof [1, 2]    // "object" (陣列也是物件)
typeof null      // "object" (這是 JS 早期的一個 Bug，為了相容性保留至今)
typeof function(){} // "function" (函式會特別顯示為 function)
```
