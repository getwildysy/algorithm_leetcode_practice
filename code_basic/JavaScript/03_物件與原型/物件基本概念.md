# 物件基本概念 (Objects Basics)

在 JavaScript 中，**物件 (Object)** 是最核心的資料結構。幾乎所有的東西在 JavaScript 中都是物件，或者可以被視為物件。

物件可以看作是一個**鍵值對 (Key-Value Pairs)** 的集合，用於儲存各種數據和功能。

## 1. 建立物件

### (1) 物件字面量 (Object Literal)

這是最常用、最簡潔的方式。使用大括號 `{}`。

```javascript
const person = {
    name: "Alice",
    age: 25,
    isStudent: false,
    "likes coding": true // 如果鍵名包含空格或特殊符號，必須用引號
};
```

### (2) 建構函式 (Constructor)

使用 `new Object()` 語法。

```javascript
const car = new Object();
car.brand = "Toyota";
car.model = "Corolla";
```

## 2. 存取屬性

### (1) 點記法 (Dot Notation)

最常用的方式，語法簡潔。

```javascript
console.log(person.name); // Alice
```

### (2) 括號記法 (Bracket Notation)

當屬性名稱**包含特殊字元**（如空格）或屬性名稱是**動態變數**時使用。

```javascript
console.log(person["likes coding"]); // true

const key = "age";
console.log(person[key]); // 25 (等同於 person.age)
```

## 3. 新增、修改與刪除屬性

```javascript
const user = { name: "Bob" };

// 新增
user.email = "bob@example.com";

// 修改
user.name = "Robert";

// 刪除
delete user.email;
```

## 4. 物件中的方法 (Methods)

當一個函式被儲存為物件的屬性時，我們稱之為**方法**。

```javascript
const calculator = {
    add: function(x, y) {
        return x + y;
    },
    // ES6 簡寫語法 (Shorthand)
    subtract(x, y) {
        return x - y;
    }
};

console.log(calculator.add(5, 3)); // 8
```

## 5. 屬性簡寫 (Property Shorthand)

如果變數名稱與物件屬性名稱相同，ES6 允許簡寫。

```javascript
const name = "Charlie";
const age = 30;

// 舊寫法
// const user = { name: name, age: age };

// ES6 簡寫
const user = { name, age };
```

## 6. Optional Chaining (可選串連) `?.`

ES2020 引入，用於安全地存取深層嵌套的屬性。如果中間某個屬性不存在 (`null` 或 `undefined`)，則直接回傳 `undefined` 而不會報錯。

```javascript
const user = {
    address: {
        city: "Taipei"
    }
};

console.log(user.address?.city); // "Taipei"
console.log(user.contact?.phone); // undefined (不會報錯)
```
