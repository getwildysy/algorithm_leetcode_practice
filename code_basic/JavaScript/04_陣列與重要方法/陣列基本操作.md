# 陣列基本操作 (Array Basics)

陣列 (Array) 是 JavaScript 中最常用的資料結構之一，用於儲存有序的資料集合。

## 1. 建立陣列

```javascript
// 1. 字面量 (推薦)
const fruits = ["Apple", "Banana", "Cherry"];

// 2. 建構函式
const numbers = new Array(1, 2, 3);
```

## 2. 存取與修改

陣列使用 **0-based** 索引 (Index)。

```javascript
const colors = ["Red", "Green", "Blue"];

console.log(colors[0]); // Red
console.log(colors[colors.length - 1]); // Blue (最後一個元素)

colors[1] = "Yellow"; // 修改
console.log(colors); // ["Red", "Yellow", "Blue"]
```

## 3. 新增與刪除元素 (最常用的 4 個方法)

這四個方法會**直接修改原陣列** (Mutable)。

### (1) `push()`: 加到尾端
回傳新的長度。

```javascript
const arr = [1, 2];
arr.push(3); 
// arr: [1, 2, 3]
```

### (2) `pop()`: 移除尾端
回傳被移除的元素。

```javascript
const last = arr.pop();
// arr: [1, 2], last: 3
```

### (3) `unshift()`: 加到開頭
回傳新的長度。效能較差，因為要移動所有元素。

```javascript
arr.unshift(0);
// arr: [0, 1, 2]
```

### (4) `shift()`: 移除開頭
回傳被移除的元素。

```javascript
const first = arr.shift();
// arr: [1, 2], first: 0
```

## 4. 擷取與合併 (不會修改原陣列)

### `slice(start, end)`
擷取陣列的一部分，回傳新陣列。包含 `start`，**不包含** `end`。

```javascript
const nums = [0, 1, 2, 3, 4];
const sub = nums.slice(1, 4); // [1, 2, 3]
const copy = nums.slice(); // 淺拷貝整個陣列
```

### `concat()`
合併兩個或多個陣列。

```javascript
const a = [1, 2];
const b = [3, 4];
const c = a.concat(b); // [1, 2, 3, 4]
```
> ES6 推薦用展開運算符 `[...a, ...b]` 取代 `concat`。

## 5. 萬能方法：`splice()`

`splice` 功能強大，可以同時進行刪除、新增、替換。**會修改原陣列**。

語法：`array.splice(start, deleteCount, item1, item2, ...)`

```javascript
const months = ['Jan', 'March', 'April', 'June'];

// 1. 插入 (在 index 1 的位置，刪除 0 個，插入 'Feb')
months.splice(1, 0, 'Feb');
// ["Jan", "Feb", "March", "April", "June"]

// 2. 刪除 (從 index 4 開始，刪除 1 個)
months.splice(4, 1);
// ["Jan", "Feb", "March", "April"]

// 3. 替換
months.splice(2, 1, 'Mar'); // 把 March 換成 Mar
```
