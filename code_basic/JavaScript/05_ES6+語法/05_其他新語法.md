# 其他 ES6+ 新語法

除了前面介紹的主要特性外，ES6+ 還引入了許多實用的語法糖和新功能。

## 1. 樣板字面量 (Template Literals)

使用反引號 (backticks) `` ` `` 取代傳統引號，支援多行字串和變數嵌入。

```javascript
const name = "Alice";
const age = 25;

// 舊寫法
// const str = "My name is " + name + " and I am " + age + " years old.";

// 新寫法
const str = `My name is ${name} and I am ${age} years old.`;

// 多行字串
const html = `
    <div>
        <h1>Title</h1>
    </div>
`;
```

## 2. 預設參數 (Default Parameters)

(已在函式章節詳細介紹，這裡複習一下)

```javascript
function greet(name = "Guest") {
    console.log(`Hello, ${name}`);
}
```

## 3. 縮寫屬性與方法 (Shorthand Properties)

(已在物件章節詳細介紹)

```javascript
const a = 1;
const obj = { 
    a, // 等同於 a: a
    hello() { } // 等同於 hello: function() {}
};
```

## 4. Nullish Coalescing Operator (`??`)

ES2020 引入。當左側運算元是 `null` 或 `undefined` 時，回傳右側運算元。
這解決了 `||` 運算符會將 `0` 或 `""` (空字串) 視為假值的問題。

```javascript
const count = 0;

// 使用 || (OR)
const num1 = count || 10; 
console.log(num1); // 10 (因為 0 被視為 falsy)

// 使用 ?? (Nullish Coalescing)
const num2 = count ?? 10;
console.log(num2); // 0 (正確保留了 0)
```

## 5. BigInt

ES2020 引入，用於表示大於 `2^53 - 1` 的整數。在數字後面加 `n`。

```javascript
const big = 123456789012345678901234567890n;
const result = big * 2n;
```

## 6. Symbol

ES6 引入的一種新的基本資料型別，表示**獨一無二**的值。常用於物件屬性的 Key，以避免名稱衝突。

```javascript
const sym1 = Symbol("desc");
const sym2 = Symbol("desc");

console.log(sym1 === sym2); // false
```
