# Day 7: 結業：分享網站並撰寫心得

恭喜你！你已經完成了從 HTML 到 CSS，從 Flexbox 到 Grid，從 Sass 到 Tailwind，最後親手打造並上線了一個個人網站。

這 8 週的旅程，你已經從一個網頁新手，變成了一位能夠獨立切版的 **Frontend Layout Master**。

## 1. 你的下一步？

CSS 只是前端的一半，接下來你可以往以下方向精進：

1.  **JavaScript 深入**：這是讓網頁有邏輯的核心。
2.  **前端框架 (React / Vue)**：現代業界標準。你學的 component 概念 (Header, Hero) 在這裡會完全對應。
3.  **接案賺錢**：你現在的技能已經足夠接簡單的靜態網頁案子了 (形象官網、活動頁)。

## 2. 撰寫 README

別忘了回頭整理你的 GitHub Repo。好的 `README.md` 是面試官看的第一樣東西。
*   **截圖**：放一張網站最美的截圖。
*   **Tech Stack**：列出你用了什麼 (Sass, BEM, Grid...)。
*   **Demo Link**：附上 Vercel 的網址。
