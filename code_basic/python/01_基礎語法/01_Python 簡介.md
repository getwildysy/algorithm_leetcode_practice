# Python 簡介

## 什麼是 Python？

Python 是一種高階、直譯式、通用且功能強大的程式語言。它以其簡潔清晰的語法而聞名，強調程式碼的可讀性和開發效率。Python 支援多種編程範式，包括物件導向、指令式和函數式編程。

## Python 的歷史

Python 由吉多·范羅蘇姆 (Guido van Rossum) 在 1980 年代末開始開發，並於 1991 年首次發布。它的設計哲學是「優雅」、「明確」、「簡單」，目標是創造一種易於閱讀和編寫的語言。

## Python 的主要特點

*   **簡潔易讀**：Python 的語法非常簡單直觀，接近自然語言，使得程式碼易於閱讀和理解，大大降低了學習門檻。它使用縮排來定義程式碼區塊，而非傳統的大括號。
*   **用途廣泛**：
    *   **網頁開發**：Django, Flask 等框架。
    *   **資料科學與機器學習**：NumPy, Pandas, Scikit-learn, TensorFlow, PyTorch 等函式庫。
    *   **自動化與腳本**：系統管理、網路配置等。
    *   **桌面應用程式**：PyQt, Tkinter。
    *   **遊戲開發**：Pygame。
    *   **教育**：因其易學性常被用於編程入門教學。
*   **豐富的標準函式庫**：Python 擁有龐大且功能強大的標準函式庫，涵蓋了各種常見的編程任務，無需額外安裝即可使用。
*   **跨平台**：Python 程式碼可以在 Windows, macOS, Linux 等多種作業系統上執行，而無需修改。
*   **開源社群**：擁有龐大活躍的開發者社群，提供豐富的資源、支援和函式庫。

## 為何學習 Python？

1.  **易學易用**：對於初學者而言，Python 是一個非常友善的入門語言。
2.  **高需求**：在資料科學、機器學習、人工智慧、網頁開發和自動化等領域，Python 的需求量持續增長。
3.  **高效開發**：簡潔的語法和豐富的函式庫讓開發人員能夠更快地實現想法。

## 第一個 Python 程式：「Hello, World!」

```python
print("Hello, World!")
```

只需一行程式碼，就可以在控制台輸出 "Hello, World!"。

## 如何執行 Python 程式碼？

1.  **Python 直譯器**：在終端機或命令提示字元中輸入 `python your_script.py` 來執行 Python 檔案。
2.  **整合開發環境 (IDE)**：使用如 VS Code, PyCharm, Jupyter Notebook 等工具，提供更豐富的開發體驗（語法高亮、自動完成、除錯等）。
