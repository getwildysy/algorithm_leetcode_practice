# 變數與資料型別 (Variables and Data Types)

在 Python 中，變數是用來儲存資料的容器。Python 是一種**動態型別 (Dynamically Typed)** 語言，這意味著你不需要在宣告變數時指定其型別，直譯器會自動根據賦予的值來判斷。

## 1. 變數 (Variables)

### 變數宣告與賦值
在 Python 中，使用等號 `=` 來進行賦值。

```python
x = 10          # 整數
name = "Alice"  # 字串
pi = 3.14       # 浮點數
is_active = True # 布林值
```

### 變數命名規則
1.  只能包含字母 (a-z, A-Z)、數字 (0-9) 和底線 (_)。
2.  **不能以數字開頭**。
3.  區分大小寫 (Case-sensitive)，`age` 和 `Age` 是不同的變數。
4.  不能使用 Python 的保留字 (Keywords)，如 `if`, `for`, `class`, `def` 等。
5.  **慣例**：Python 變數通常使用 **snake_case** (小寫字母加底線)，例如 `user_name`, `total_count`。

```python
# 合法
my_var = 1
_private = 2
var2 = 3

# 非法
# 2var = 1     (數字開頭)
# my-var = 1   (包含連字號)
# class = 1    (保留字)
```

## 2. 基本資料型別 (Data Types)

Python 中常見的基本資料型別如下：

### (1) 數值 (Numbers)
*   **Integer (整數 `int`)**：不帶小數點的數字，長度不受限制（視記憶體而定）。
*   **Float (浮點數 `float`)**：帶有小數點的數字。

```python
a = 100
b = -5
c = 3.14
d = 1.2e3 # 科學記號 (1200.0)
```

### (2) 字串 (String `str`)
由單引號 `'` 或雙引號 `"` 包圍的文字序列。

```python
s1 = 'Hello'
s2 = "Python"
s3 = "123" # 這也是字串
```

### (3) 布林值 (Boolean `bool`)
只有兩個值：`True` (真) 和 `False` (假)。常用於條件判斷。注意首字母必須大寫。

```python
is_login = True
has_error = False
```

### (4) NoneType
只有一個值 `None`，代表「空值」或「無」。類似於其他語言的 null。

```python
result = None
```

## 3. 檢查型別 (`type()`)

使用 `type()` 函式可以查看變數的資料型別。

```python
x = 10
print(type(x)) # <class 'int'>

y = "Hello"
print(type(y)) # <class 'str'>
```

## 4. 型別轉換 (Type Conversion)

Python 提供了內建函式來進行型別轉換。

*   `int(x)`: 轉為整數 (無條件捨去小數部分)。
*   `float(x)`: 轉為浮點數。
*   `str(x)`: 轉為字串。
*   `bool(x)`: 轉為布林值。通常 0, 空字串, None, 空列表等會轉為 `False`，其餘為 `True`。

```python
# 字串轉數字
num_str = "123"
num = int(num_str)
print(num + 1) # 124

# 數字轉字串
age = 25
msg = "I am " + str(age) + " years old."

# 浮點數轉整數
print(int(3.99)) # 3

# 布林轉換
print(bool(0))       # False
print(bool("Hello")) # True
```
