# 運算子 (Operators)

運算子是用於對變數和值執行操作的符號。

## 1. 算術運算子 (Arithmetic Operators)

用於執行數學運算。

| 運算子 | 描述 | 範例 (a=10, b=3) | 結果 |
| :--- | :--- | :--- | :--- |
| `+` | 加法 | `a + b` | 13 |
| `-` | 減法 | `a - b` | 7 |
| `*` | 乘法 | `a * b` | 30 |
| `/` | 除法 | `a / b` | 3.333... (總是回傳浮點數) |
| `//` | **整數除法** (地板除) | `a // b` | 3 (無條件捨去小數) |
| `%` | **取餘數** (模運算) | `a % b` | 1 |
| `**` | **次方** (指數) | `a ** 2` | 100 (10 的 2 次方) |

```python
print(10 / 3)  # 3.3333333333333335
print(10 // 3) # 3
print(10 % 3)  # 1
print(2 ** 3)  # 8
```

## 2. 比較運算子 (Comparison Operators)

用於比較兩個值，回傳布林值 (`True` 或 `False`)。

| 運算子 | 描述 | 範例 |
| :--- | :--- | :--- |
| `==` | 等於 | `5 == 5` (True) |
| `!=` | 不等於 | `5 != 3` (True) |
| `>` | 大於 | `5 > 3` (True) |
| `<` | 小於 | `5 < 3` (False) |
| `>=` | 大於等於 | `5 >= 5` (True) |
| `<=` | 小於等於 | `4 <= 5` (True) |

## 3. 邏輯運算子 (Logical Operators)

用於結合條件判斷。

*   **`and`**: 且。兩邊都為 True 才回傳 True。
*   **`or`**: 或。只要有一邊為 True 就回傳 True。
*   **`not`**: 非。反轉布林值。

```python
x = 5
print(x > 3 and x < 10) # True
print(x > 3 or x < 4)   # True
print(not (x > 3))      # False
```

## 4. 賦值運算子 (Assignment Operators)

用於將值賦給變數，常與算術運算子結合。

```python
x = 10
x += 5  # 等同於 x = x + 5 (x 變 15)
x -= 2  # 等同於 x = x - 2
x *= 2  # 等同於 x = x * 2
x /= 2  # 等同於 x = x / 2
```

## 5. 成員運算子 (Membership Operators)

用於檢查序列 (如字串、列表、Tuple) 中是否包含某個值。

*   **`in`**: 存在則回傳 True。
*   **`not in`**: 不存在則回傳 True。

```python
fruits = ["apple", "banana"]
print("apple" in fruits)    # True
print("orange" not in fruits) # True
```

## 6. 身份運算子 (Identity Operators)

用於比較兩個物件是否指向**同一個記憶體位址**。

*   **`is`**: 是同一個物件。
*   **`is not`**: 不是同一個物件。

> **注意**：`==` 比較的是**值**是否相等，`is` 比較的是**物件**是否相同。

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b) # True (值相同)
print(a is b) # False (不同的記憶體物件)
print(a is c) # True (指向同一個物件)
```
