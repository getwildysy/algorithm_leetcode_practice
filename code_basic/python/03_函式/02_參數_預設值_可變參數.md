# 參數：預設值與可變參數

Python 提供了非常靈活的參數定義方式，包括位置參數、關鍵字參數、預設參數以及可變長度參數。

## 1. 位置參數 (Positional Arguments)

最基本的參數傳遞方式，依照順序對應。

```python
def describe_pet(animal_type, pet_name):
    print(f"I have a {animal_type} named {pet_name}.")

describe_pet('hamster', 'Harry')
```

## 2. 關鍵字參數 (Keyword Arguments)

呼叫時指定參數名稱，順序就不重要了。

```python
describe_pet(pet_name='Harry', animal_type='hamster')
```

## 3. 預設參數 (Default Values)

在定義時給參數一個預設值。如果呼叫時沒有提供該參數，則使用預設值。
**注意：預設參數必須放在非預設參數的後面。**

```python
def greet(name, msg="Good morning"):
    print(f"Hello {name}, {msg}")

greet("Alice")                  # 使用預設 msg
greet("Bob", "How are you?")    # 覆蓋預設 msg
```

### 重要陷阱：可變物件作為預設值

**永遠不要**使用 List 或 Dict 等可變物件作為預設參數值。因為預設值只會在函式定義時被計算一次，之後會被共用。

```python
# 錯誤寫法
def append_to(num, target=[]):
    target.append(num)
    return target

print(append_to(1)) # [1]
print(append_to(2)) # [1, 2] (驚訝！它記住了上次的狀態)

# 正確寫法
def append_to_safe(num, target=None):
    if target is None:
        target = []
    target.append(num)
    return target
```

## 4. 可變參數 (Arbitrary Arguments)

### `*args` (非關鍵字可變參數)

接收任意數量的位置參數，函式內部會將其視為一個 **Tuple**。

```python
def sum_all(*numbers):
    total = 0
    for n in numbers:
        total += n
    return total

print(sum_all(1, 2, 3, 4)) # 10
```

### `**kwargs` (關鍵字可變參數)

接收任意數量的關鍵字參數，函式內部會將其視為一個 **Dictionary**。

```python
def print_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30, city="Taipei")
```

## 5. 參數順序

如果混合使用，參數定義的順序必須是：
1.  位置參數
2.  `*args`
3.  預設參數 (Keyword-only)
4.  `**kwargs`

```python
def complex_func(a, b, *args, option=True, **kwargs):
    pass
```
