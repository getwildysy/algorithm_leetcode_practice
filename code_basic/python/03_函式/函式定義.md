# Python 函式定義 (Function Definition)

函式是組織好的、可重複使用的程式碼區塊，用於執行單一的、相關聯的動作。使用函式可以提高應用程式的模組化程度和程式碼的重複利用率。

## 1. 基本語法

使用 `def` 關鍵字定義函式。

```python
def function_name(parameters):
    """Docstring: 函式的說明文件 (可選)"""
    # 函式本體
    # ...
    return value # (可選)
```

### 範例

```python
def say_hello(name):
    """這是一個打招呼的函式"""
    print(f"Hello, {name}!")

say_hello("Alice") # 呼叫函式
```

## 2. Docstrings (文件字串)

函式本體的第一行如果是字串，會被視為 Docstring，用於說明函式的用途。可以透過 `help(function_name)` 或 `function_name.__doc__` 查看。

```python
def add(a, b):
    """回傳兩個數字的總和。"""
    return a + b

print(add.__doc__)
```

## 3. 函式作為物件 (First-class Object)

在 Python 中，函式是一等公民，可以像變數一樣被傳遞。

```python
def shout(text):
    return text.upper()

def whisper(text):
    return text.lower()

# 高階函式：接受函式作為參數
def greet(func):
    greeting = func("Hi There")
    print(greeting)

greet(shout)   # "HI THERE"
greet(whisper) # "hi there"
```

## 4. 空函式

如果你需要定義一個尚未實作的函式（例如為了先設計結構），可以使用 `pass` 語句。

```python
def todo_function():
    pass
```
