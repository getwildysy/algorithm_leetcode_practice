# 自訂模組與套件

## 1. 建立自訂模組

只需建立一個 `.py` 檔案即可。

假設我們建立一個 `calculator.py`：

```python
# calculator.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

PI = 3.14159
```

在另一個檔案中使用它：

```python
# main.py
import calculator

print(calculator.add(5, 3))
print(calculator.PI)
```

## 2. 套件 (Package)

套件是用於組織模組的**資料夾**。
在 Python 3.3 之前，資料夾內必須包含一個 `__init__.py` 檔案才能被視為套件。雖然現在不是強制，但保留它仍是好習慣，可以用來初始化套件或控制匯出內容。

### 目錄結構範例

```
my_project/
    main.py
    utils/              <-- 套件
        __init__.py
        string_utils.py
        math_utils.py
```

### 使用套件

```python
# main.py

# 方法 1
import utils.math_utils
result = utils.math_utils.add(1, 2)

# 方法 2
from utils import string_utils
print(string_utils.uppercase("hello"))

# 方法 3 (搭配別名)
import utils.math_utils as math
print(math.add(1, 2))
```

## 3. `__init__.py` 的作用

當匯入套件時，`__init__.py` 會自動執行。我們可以在這裡簡化匯入路徑。

如果在 `utils/__init__.py` 中寫入：

```python
# utils/__init__.py
from .math_utils import add
from .string_utils import uppercase
```

那麼在 `main.py` 中就可以直接從套件匯入：

```python
# main.py
from utils import add, uppercase # 直接從 utils 匯入，不用管內部結構

print(add(1, 2))
```
