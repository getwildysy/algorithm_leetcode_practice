# 類別 (Class) 與 物件 (Object)

Python 是一門物件導向程式設計 (OOP) 語言。幾乎所有的東西在 Python 中都是物件，擁有屬性和方法。

## 1. 什麼是類別與物件？

*   **類別 (Class)**：就像是藍圖或模具，定義了物件應該有的屬性 (資料) 和方法 (行為)。
*   **物件 (Object)**：是根據類別建立出來的實體 (Instance)。

舉例來說，「汽車設計圖」是類別，而根據設計圖製造出來的「每一輛真實的汽車」就是物件。

## 2. 定義類別

使用 `class` 關鍵字。類別名稱通常採用 **PascalCase** (首字母大寫) 命名慣例。

```python
class Dog:
    # 類別屬性 (Class Attribute)
    species = "Canis familiaris"

    # 建構子 (Constructor)
    def __init__(self, name, age):
        # 實體屬性 (Instance Attributes)
        self.name = name
        self.age = age

    # 實體方法 (Instance Method)
    def bark(self):
        return f"{self.name} says Woof!"
```

## 3. 建立物件 (實體化)

呼叫類別名稱並傳入必要的參數。

```python
buddy = Dog("Buddy", 9)
miles = Dog("Miles", 4)
```

## 4. 存取屬性與方法

使用點號 `.` 來存取。

```python
print(buddy.name)    # Buddy
print(buddy.species) # Canis familiaris
print(miles.bark())  # Miles says Woof!
```

## 5. `self` 是什麼？

在類別的方法定義中，第一個參數通常命名為 `self`。它代表**物件本身**。
當你呼叫 `buddy.bark()` 時，Python 會自動將 `buddy` 物件作為第一個參數傳入 `bark` 方法，因此：
`Dog.bark(buddy)` 等同於 `buddy.bark()`。
