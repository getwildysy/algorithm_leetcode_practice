# 屬性 (Attributes) 與 方法 (Methods)

在 Python 的類別中，成員主要分為屬性和方法。

## 1. 屬性 (Attributes)

### 實體屬性 (Instance Attributes)
屬於個別物件的變數，通常在 `__init__` 中使用 `self.name = value` 定義。每個物件的實體屬性互不影響。

### 類別屬性 (Class Attributes)
屬於類別本身的變數，在方法之外定義。所有物件共享同一個類別屬性。

```python
class Circle:
    pi = 3.14159 # 類別屬性
    
    def __init__(self, radius):
        self.radius = radius # 實體屬性

c1 = Circle(10)
c2 = Circle(20)

print(c1.radius) # 10
print(c2.radius) # 20
print(c1.pi)     # 3.14159
print(Circle.pi) # 3.14159 (推薦透過類別存取)
```

## 2. 方法 (Methods)

### 實體方法 (Instance Methods)
最常見的方法類型。第一個參數是 `self`，可以存取實體屬性和其他實體方法。

```python
    def area(self):
        return self.pi * (self.radius ** 2)
```

### 類別方法 (Class Methods)
使用 `@classmethod` 裝飾器。第一個參數通常命名為 `cls`，代表類別本身。無法存取實體屬性，但可以存取類別屬性。常用於定義**工廠方法**。

```python
class Date:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day

    @classmethod
    def from_string(cls, date_string):
        # 解析 "2023-10-27"
        year, month, day = map(int, date_string.split('-'))
        return cls(year, month, day) # 回傳新物件

d = Date.from_string("2023-10-27")
```

### 靜態方法 (Static Methods)
使用 `@staticmethod` 裝飾器。不需要 `self` 或 `cls` 參數。它們就像是普通函式，只是剛好放在類別裡面方便組織。通常用於不需要存取物件或類別狀態的工具函式。

```python
class MathUtils:
    @staticmethod
    def add(x, y):
        return x + y

print(MathUtils.add(5, 3))
```
