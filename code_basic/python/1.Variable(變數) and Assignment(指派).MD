一、命名原則 1.必須由英文字母、\_(底線)或中文字開頭，建議使用英文字母。

2.變數名稱只能有英文字母、\_(底線)或中文字所組成。

3.英文大小寫視為不同。

4.系統保留字或內建函數名稱不可用來當作是變數稱。

二、基本數學運算
1、 + - \* /

2、0 求餘數: " %" ; 求 商數: " // "。

3、次方: " \*\* "

1、Slincing

```python
#string[start(inclusive),end(exclusive),step size(optional)]

x = "abcdefgh"
print(x[2:]) # cdefgh
print(x[:3]) # abc
print(x[1:4])# bcd
print(x[1:5:2]) # bd
print(x[::-1]) # hgfedcba
```
