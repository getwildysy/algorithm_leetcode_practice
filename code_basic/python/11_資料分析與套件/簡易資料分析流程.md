# 簡易資料分析流程 (Data Analysis Pipeline)

一個典型的資料分析專案通常包含以下步驟：

1.  **資料獲取 (Data Acquisition)**: 從檔案、API 或資料庫讀取資料。
2.  **資料清理 (Data Cleaning)**: 處理缺漏值、重複值、異常值。
3.  **探索性資料分析 (EDA)**: 了解資料分佈、相關性。
4.  **特徵工程 (Feature Engineering)**: 轉換資料格式，提取有用的特徵。
5.  **模型訓練/分析**: (如果是機器學習專案)
6.  **視覺化與報告**: 呈現結果。

## 範例：分析學生成績

假設我們有一個 `grades.csv`。

### 步驟 1 & 2: 讀取與清理

```python
import pandas as pd
import numpy as np

# 模擬資料
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David', 'Eve'],
    'Math': [85, 70, 90, np.nan, 60], # 有缺漏值
    'English': [80, 85, 88, 92, 75]
}
df = pd.DataFrame(data)

# 填補缺漏值 (用平均值填補)
df['Math'] = df['Math'].fillna(df['Math'].mean())

print("清理後的資料:")
print(df)
```

### 步驟 3 & 4: 分析與特徵工程

```python
# 計算總分與平均
df['Total'] = df['Math'] + df['English']
df['Average'] = df['Total'] / 2

# 找出表現最好的學生
top_student = df.loc[df['Total'].idxmax()]
print(f"\n第一名是: {top_student['Name']}, 總分: {top_student['Total']}")
```

### 步驟 6: 視覺化

```python
import matplotlib.pyplot as plt

plt.figure(figsize=(8, 5))
plt.bar(df['Name'], df['Total'], color='skyblue')
plt.title("Student Total Scores")
plt.xlabel("Name")
plt.ylabel("Total Score")
plt.show()
```
