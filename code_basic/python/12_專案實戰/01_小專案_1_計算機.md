# 小專案 1：簡易計算機

**目標**: 練習基本的函式定義、使用者輸入處理 (`input`)、條件判斷 (`if/elif`) 以及迴圈控制。

## 程式碼實作

```python
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    if y == 0:
        return "Error! Division by zero."
    return x / y

def calculator():
    print("=== 簡易計算機 ===")
    print("1. 加法 (+)")
    print("2. 減法 (-)")
    print("3. 乘法 (*)")
    print("4. 除法 (/)")
    print("q. 離開")

    while True:
        choice = input("\n請選擇運算 (1/2/3/4/q): ")

        if choice.lower() == 'q':
            print("感謝使用，再見！")
            break

        if choice in ('1', '2', '3', '4'):
            try:
                num1 = float(input("請輸入第一個數字: "))
                num2 = float(input("請輸入第一個數字: "))
            except ValueError:
                print("輸入錯誤，請輸入有效的數字！")
                continue

            if choice == '1':
                print(f"{num1} + {num2} = {add(num1, num2)}")
            elif choice == '2':
                print(f"{num1} - {num2} = {subtract(num1, num2)}")
            elif choice == '3':
                print(f"{num1} * {num2} = {multiply(num1, num2)}")
            elif choice == '4':
                result = divide(num1, num2)
                if result == "Error! Division by zero.":
                    print(result)
                else:
                    print(f"{num1} / {num2} = {result}")
        else:
            print("無效的輸入，請重新選擇。")

if __name__ == "__main__":
    calculator()
```
