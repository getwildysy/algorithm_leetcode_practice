# 常見錯誤與除錯技巧

## 1. 常見錯誤 (Errors)

### `SyntaxError` (語法錯誤)
程式碼不符合 Python 語法規則，通常是漏了括號、冒號或引號。

```python
# SyntaxError: expected ':'
if True 
    print("Hello")
```

### `IndentationError` (縮排錯誤)
Python 嚴格要求縮排，通常是因為空格和 Tab 混用，或縮排層級不對。

```python
def func():
print("Error") # IndentationError: expected an indented block
```

### `NameError` (名稱錯誤)
使用了未定義的變數。

```python
print(age) # NameError: name 'age' is not defined
```

### `TypeError` (型別錯誤)
對錯誤的型別進行了操作。

```python
print("Age: " + 25) # TypeError: can only concatenate str (not "int") to str
```

### `IndexError` / `KeyError`
存取了不存在的索引 (List) 或 鍵 (Dict)。

## 2. 除錯技巧 (Debugging)

### (1) 使用 `print()`
最簡單直接的方法。在關鍵位置印出變數的值，確認程式執行流程。

### (2) 閱讀錯誤訊息 (Traceback)
不要害怕紅色的錯誤訊息。
1.  看**最後一行**：確認錯誤類型和訊息。
2.  看**檔案路徑與行號**：找到錯誤發生的位置。

### (3) 使用 VS Code 偵錯工具 (Debugger)
1.  在行號左側點擊設定**斷點 (Breakpoint)** (紅點)。
2.  按下 `F5` 開始偵錯。
3.  程式會停在斷點處，你可以查看變數狀態，並單步執行 (`Step Over`, `Step Into`)。
