# 安裝與環境配置

## 1. 下載與安裝 Python

前往 Python 官方網站 [python.org](https://www.python.org/downloads/) 下載適合你作業系統 (Windows, macOS, Linux) 的最新版本。

**Windows 安裝注意事項**:
在安裝過程中，務必勾選 **"Add Python to PATH"** (將 Python 加入環境變數)。這讓你可以直接在命令提示字元 (cmd) 中輸入 `python` 來執行。

## 2. 驗證安裝

開啟終端機 (Terminal) 或命令提示字元 (cmd)，輸入：

```bash
python --version
# 或
python3 --version
```

如果有顯示版本號 (如 `Python 3.10.x`)，代表安裝成功。

## 3. VS Code 設定

Visual Studio Code 是目前最受歡迎的 Python 編輯器。

1.  **安裝 VS Code**: [code.visualstudio.com](https://code.visualstudio.com/)
2.  **安裝 Python 擴充套件**:
    *   開啟 VS Code。
    *   點擊左側的擴充套件圖示 (Extensions)。
    *   搜尋 "Python" (Microsoft 出品)。
    *   點擊 Install。

## 4. 選擇直譯器

在 VS Code 中開啟 Python 檔案 (`.py`) 後，點擊右下角的狀態列 (顯示 Python 版本的地方)，可以切換專案使用的 Python 直譯器 (例如切換到虛擬環境)。
