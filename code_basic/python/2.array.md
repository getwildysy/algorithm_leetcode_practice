# Python 陣列 (Array/List)

在 Python 中，最常用的「陣列」型別其實是 **列表 (List)**。它是一個有序、可變的集合，可以容納不同型別的資料。

## 一、 建立列表 (List)

使用中括號 `[]` 來建立列表。

```python
# 空列表
empty_list = []

# 包含資料的列表
numbers = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "orange"]
mixed = [1, "hello", 3.14, True]
```

## 二、 讀取與修改

### 1. 索引 (Indexing)
使用索引值來存取特定元素，索引從 0 開始。

```python
colors = ["red", "green", "blue"]
print(colors[0]) # red
print(colors[-1]) # blue (最後一個)
```

### 2. 切片 (Slicing)
語法：`list[start:end:step]`

```python
nums = [0, 1, 2, 3, 4, 5]
print(nums[1:4]) # [1, 2, 3] (包含 1，不包含 4)
print(nums[:3])  # [0, 1, 2] (從頭開始)
print(nums[::2]) # [0, 2, 4] (每隔一個)
```

### 3. 修改元素
列表是**可變的 (Mutable)**，可以直接修改。

```python
colors[1] = "yellow"
print(colors) # ['red', 'yellow', 'blue']
```

## 三、 常用方法 (Methods)

### 1. 新增元素
*   `append(x)`: 加到最後面。
*   `insert(i, x)`: 插在指定位置 `i`。

```python
arr = [1, 2]
arr.append(3)    # [1, 2, 3]
arr.insert(0, 0) # [0, 1, 2, 3]
```

### 2. 刪除元素
*   `remove(x)`: 刪除第一個值為 `x` 的元素。
*   `pop(i)`: 移除並回傳索引 `i` 的元素 (預設為最後一個)。

```python
arr.remove(2) # [0, 1, 3]
last = arr.pop() # 3, arr 變為 [0, 1]
```

### 3. 其他
*   `len(list)`: 取得長度。
*   `sort()`: 排序。
*   `reverse()`: 反轉。

```python
a = [3, 1, 2]
a.sort() # [1, 2, 3]
print(len(a)) # 3
```
