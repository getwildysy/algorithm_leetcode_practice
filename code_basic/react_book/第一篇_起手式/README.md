✅ 建立紮實的 JavaScript 基礎

## 常見問題

**Q: 我沒有 JavaScript 基礎，可以直接學 React 嗎？**  
A: 強烈建議先具備基本的 JavaScript 知識。本書第 2 章有提供精簡的 JS 補給，但如果完全沒有基礎，建議先學習 JavaScript 基礎課程。

**Q: 為什麼不使用 Create React App？**  
A: Vite 提供更快的開發體驗和更好的效能，且是目前社群推薦的選擇。CRA 已經逐漸被淘汰。

**Q: 需要學習 Class Component 嗎？**  
A: 不需要。現代 React 開發已經完全採用 Functional Component + Hooks，除非你需要維護舊專案。
