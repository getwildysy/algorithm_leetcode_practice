# 實戰演練：製作一個「個人資訊卡 (Profile Card)」元件，透過 Props 傳入資料

## 專案目標

製作一個可重用的個人資訊卡元件，展示以下資訊：
- 頭像
- 姓名
- 職稱
- 個人簡介
- 社交媒體連結

## 功能需求

### 基本需求
- [ ] 顯示使用者頭像
- [ ] 顯示姓名與職稱
- [ ] 顯示個人簡介
- [ ] 支援透過 Props 傳入資料

### 進階需求
- [ ] 支援預設頭像
- [ ] 支援社交媒體圖示
- [ ] 響應式設計
- [ ] 懸停效果

## 元件結構

```jsx
<ProfileCard
  avatar="url"
  name="Alice Chen"
  title="Frontend Developer"
  bio="熱愛 React 開發"
  social={{
    github: "alice",
    twitter: "alice"
  }}
/>
```

- 建立 ProfileCard.css

- 加入視覺效果

### 步驟 3：處理 Props
- 解構 Props
- 設定預設值
- 條件渲染

### 步驟 4：測試與優化
- 測試不同資料
- 處理邊界情況
- 優化使用者體驗

完成後應該能夠：
- ✅ 顯示完整的個人資訊卡
