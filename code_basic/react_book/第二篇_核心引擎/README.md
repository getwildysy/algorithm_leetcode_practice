## 常見陷阱

⚠️ **狀態更新不即時**：記住 setState 是異步的  
⚠️ **useEffect 無限迴圈**：依賴陣列設定錯誤  
⚠️ **忘記清理副作用**：導致記憶體洩漏  
⚠️ **用 index 當 key**：會導致渲染錯誤

- 狀態管理（useState）
- 表單處理（受控元件）
- 列表渲染（map + key）
- 資料持久化（useEffect + localStorage）
