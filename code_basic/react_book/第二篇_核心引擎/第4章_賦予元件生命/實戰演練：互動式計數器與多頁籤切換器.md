# 實戰演練：互動式計數器與多頁籤切換器 (Tabs)

## 專案一：互動式計數器

### 功能需求
- [ ] 顯示當前數值
- [ ] 增加按鈕（+1）
- [ ] 減少按鈕（-1）
- [ ] 重置按鈕
- [ ] 步進值設定（可選）
- [ ] 最大/最小值限制（可選）

### 核心程式碼框架

```jsx
function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <h1>計數: {count}</h1>
      <button onClick={() => setCount(count + 1)}>+</button>
      <button onClick={() => setCount(count - 1)}>-</button>
      <button onClick={() => setCount(0)}>重置</button>
    </div>
  );
}
```

## 專案二：多頁籤切換器

### 功能需求
- [ ] 多個頁籤選項
- [ ] 點擊切換內容
- [ ] 高亮當前選中頁籤
- [ ] 顯示對應內容
- [ ] 支援鍵盤導航（可選）

### 核心程式碼框架

```jsx
function Tabs() {
  const [activeTab, setActiveTab] = useState('tab1');
  
  const tabs = [
    { id: 'tab1', label: '頁籤 1', content: '內容 1' },
    { id: 'tab2', label: '頁籤 2', content: '內容 2' },
    { id: 'tab3', label: '頁籤 3', content: '內容 3' }
  ];
  
  return (
    <div>
      <div className="tab-buttons">
        {tabs.map(tab => (
          <button
            key={tab.id}
            className={activeTab === tab.id ? 'active' : ''}
            onClick={() => setActiveTab(tab.id)}
          >
            {tab.label}
          </button>
        ))}
      </div>
      <div className="tab-content">
        {tabs.find(t => t.id === activeTab)?.content}
      </div>
    </div>
  );
}
```
