## 實戰專案建議

### 完整部落格系統
整合所有學過的知識，建立一個功能完整的部落格：

**前台功能**
- 文章列表（分頁、篩選）
- 文章詳情（動態路由）
- 分類與標籤
- 搜尋功能
- 深色模式

**後台功能**
- 登入驗證（路由守衛）
- 文章管理（CRUD）
- 圖片上傳
- 草稿儲存

**技術棧**
- React Router（路由）
- Zustand（狀態管理）
- TanStack Query（API 管理）
- Tailwind CSS（樣式）
- React Hook Form（表單）
